使用vue-cli3开发chrome插件

前言

  • 安装nodejs
  • 安装vue-cli3，npm install -g @vue/cli，即可使用vue命令

搭建环境

  • 创建一个vue-cli3项目： vue create vue-extension，对话流程选择默认就行。
  • 进入项目cd vue-extension
  • 安装vue-cli-plugin-chrome-ext插件：vue add chrome-ext,根据安装对话选项设置好。

引入element UI

  • npm install element-ui

chrome的页面展示在src/popup里

  • 在src/popup/index.js中引入elementui
  • 修改``src/popup/App/App.vue`展示页面

运行项目

  • npm run build-watch运行开发环境，根目录生成dist文件夹，chrome加载dist文件夹即可
